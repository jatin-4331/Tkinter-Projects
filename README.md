# 🧮 Tkinter Calculator App

A simple **Calculator Application** built using **Python Tkinter**.
This project performs basic arithmetic operations with a graphical user interface.

## 🚀 Features

* User-friendly GUI
* Basic arithmetic operations:

  * Addition (+)
  * Subtraction (-)
  * Multiplication (*)
  * Division (/)
* Clear button to reset calculations
* Error handling for invalid inputs

## 🛠️ Technologies Used

* Python
* Tkinter (Python GUI library)

## ▶️ How to Run the Project

1. Install Python (if not installed)
2. Clone the repository

```
git clone https://github.com/jatin-4331/Tkinter-Projects.git
```

3. Navigate to the project folder

```
cd Tkinter-Projects
```

4. Run the program

```
python Calculator.py
```
